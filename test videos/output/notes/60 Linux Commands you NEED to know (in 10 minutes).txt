**Essential Linux Commands**

**SSH**
* Connect to a Linux machine: `ssh [user]@[server]`

**File Management**

* List files: `ls`
* List files with details: `ls -l`
* List hidden files: `ls -a`
* Print working directory: `pwd`
* Change directory: `cd [directory]`
* Create a file: `touch [file]`
* Create multiple files: `touch [file1] [file2] ...`
* Create a file in the future: `touch -d [date] [file]`
* Display the contents of a file: `cat [file]`
* Edit a file:
    * `nano [file]`
    * `vim [file]` (for advanced users)
* Shred a file: `shred [file]`
* Create a directory: `mkdir [directory]`
* Copy a file: `cp [source] [destination]`
* Move a file: `mv [source] [destination]`
* Remove a file: `rm [file]`
* Remove a directory: `rmdir [directory]` (if empty)
* Remove a non-empty directory: `rm -r [directory]`
* Create a symbolic link: `ln -s [source] [link]`

**System Maintenance**

* Clean the terminal: `clear`
* Get system information: `whoami`
* Create a new user (requires sudo): `adduser [username]`
* Change password: `passwd [username]`
* Switch users: `su [username]`
* Exit a command: `exit`
* Install packages (Debian/Ubuntu): `sudo apt install [package]`
* Install packages (Red Hat/CentOS): `sudo yum install [package]`
* Update repositories: `sudo apt update`
* Get help on a command: `man [command]`
* Get a quick description of a command: `whatis [command]`
* Find the location of a command: `which [command]`

**Internet and Networking**

* Download a file:
    * `wget [URL]`
    * `curl [URL] > [file]`
* Unzip a file: `unzip [file]`
* Read a file page by page: `less [file]`
* View the beginning of a file: `head [file]`
* View the end of a file: `tail [file]`
* Compare two files: `cmp [file1] [file2]`
* Find text in files: `find [directory] -name [regex]`

**Utilities**

* Sort a file: `sort [file]`
* Change file permissions: `chmod [permission] [file]`
* Change file ownership: `chown [user] [file]`

**Networking**

* Get IP address: `ifconfig`
* Get IP address for a specific interface: `ifconfig [interface] | grep 'inet addr'`
* Get DNS settings: `cat /etc/resolv.conf`
* Check website availability: `ping [website]`
* Trace the path to a website: `traceroute [website]`
* Check open ports: `netstat -ntlp`
* Allow traffic on a port: `ufw allow [port]`
* Enable firewall: `ufw enable`

**System Information**

* Get system information:
    * `uname`
    * `uname -a`
* Get a formatted system report: `neofetch`

**Other**

* Get a calendar: `cal`
* Perform calculations: `echo [expression] | bc`
* Check memory usage: `free`
* Check disk space: `df -H`

**Process Management**

* List processes: `ps aux`
* Monitor running processes: `top`
* Monitor running processes in a graphical interface: `htop`
* Kill a process: `kill [signal] [process ID]`
* Kill a process by name: `pkill -f [process name]`
* Start/stop/restart services: `systemctl [action] [service]`

**History and Shutdown**

* Show command history: `history`
* Reboot the system: `reboot`
* Shut down the system: `shutdown`
* Shut down the system immediately: `shutdown -H now`
